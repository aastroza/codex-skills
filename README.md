# Codex Skills

Shared Codex skills, organized by domain.

## Skills

### Magic: The Gathering

- `mtga-control`: Controls Magic: The Gathering Arena on macOS with screenshots, coordinate clicks, drag-and-drop card play, and a CoreGraphics fallback for MTGA UI that ignores normal synthetic clicks.

- `scryfall-api`: Calls the Scryfall REST API for exact and fuzzy card lookup, search syntax queries, rulings, set data, legality, decklist verification, and mana-base searches that account for typed lands such as shocklands.

### Writing

- `write-good-docs`: Writes and revises READMEs, project guides, engineering blog posts, setup docs, and agent instructions for code projects, with a focus on useful structure, concrete context, and less AI-sounding prose.

- `write-with-aida`: Shapes, reviews, and rewrites marketing, sales, educational, consulting, newsletter, landing page, social, or thought-leadership drafts using the AIDA framework: Attention, Interest, Desire, Action.

- `write-with-zinsser`: Drafts, edits, and critiques prose with a clear, warm, source-faithful nonfiction standard, plus a scanner for common AI-writing tells. Based on Isaac Flath's [`writing-with-zinsser`](https://github.com/Isaac-Flath/agent-starter-wiki/blob/main/writing-with-zinsser.md) skill and Wikipedia's [`Signs of AI writing`](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing).
